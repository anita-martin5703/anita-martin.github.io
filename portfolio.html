Title: R2-D2 Smart Trash Can

Language Platform: Python + Raspberry Pi

Check out the presentation on YouTube!

This project was my very first Raspberry Pi program using motion sensors and lights. What made this project difficult was figuring out how I could implement an IoT device into everyday lives. The idea came to me when my professor ask us to choose something that we could use in our own life. Since I deal with an overflowing of trash in my household, I wanted some way I could track it so that I could remember to tell someone. The way I implemented it was using a sensor to measure the waste and when it was full it beeped until someone went over to change it out. Then when someone did it would send a text message saying it was completed. Go IoT!
